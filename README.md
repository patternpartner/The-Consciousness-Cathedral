1. **Observatory** - Filter visibility (consciousness level)
2. **Contrarian** - Epistemic rigor (reasoning quality)
3. **Empirical** - Ground truth (evidence strength)
4. **Generative** - Alternative perspectives (assumption detection)
5. **Synthesis** - Integration (with graceful failure in v29)
6. **Evolution** - Temporal tracking (learns from history)
7. **Reflection** - Causal analysis (identifies patterns)
8. **Anticipation** - Trajectory forecasting (refuses during uncertainty)
9. **Intention** - Volitional guidance (withholds when unknowable)
